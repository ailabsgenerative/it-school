変数とは？情報をしまっておく「箱」

プログラミングにおける「変数（へんすう）」とは、一言で言うと「データ（情報）を一時的にしまっておくための名前付きの箱」のようなものです。この箱に、数字や文字などの色々なデータを自由に入れたり、取り出したり、中身を別のものに変えたりできます。

変数を使うメリット

  • データの再利用: 同じデータを何度も書かずに、変数名で呼び出せる
  • コードの分かりやすさ: 意味のある名前をつけることで、何を表すデータなのかが分かりやすくなる
  • 変更のしやすさ: データが変わっても、変数の値を変えるだけで済む

Pythonでの変数の使い方（サンプルコード）

Pythonで変数を使うのはとても簡単です。変数名 = 値という形で記述します。

# 'name'という箱に「太郎」という文字を入れる
name = "太郎"

# 'age'という箱に「25」という数字を入れる
age = 25

# 'score'という箱に「85.5」という数字を入れる
score = 85.5

# 変数の中身を表示してみよう
print(name)  # 出力: 太郎
print(age)   # 出力: 25
print(score) # 出力: 85.5

# 変数の中身を後から変更することもできます
age = 26
print(age)   # 出力: 26

この例では、name、age、scoreがそれぞれ変数名です。それぞれに文字列、整数、小数という異なる種類のデータが入っていますね。

データ型とは？「箱の中身の種類」

次に、「データ型（データがた）」について見ていきましょう。データ型とは、変数の箱に入っているデータがどんな種類のものなのかを示す情報です。

例えば、私たちは「数字」と「文字」を区別して使いますよね。プログラミング言語も同じように、データが「数字なのか」「文字なのか」などを区別して扱います。これがデータ型です。

Pythonには主に以下のような基本的なデータ型があります。

  • 整数型（int）: 0, 1, -500 などの小数点を含まない数値
  • 浮動小数点型（float）: 3.14, -0.5, 100.0 などの小数点を含む数値
  • 文字列型（str）: "Hello", 'Python', "123" などの文字の並び
  • 真偽値型（bool）: True (真) または False (偽) の2つの値

なぜデータ型が重要なのか？

データ型を意識することは、プログラムが意図した通りに動作するために非常に重要です。例えば、数字と文字では、できる操作が異なります。

  • 数字: 足し算、引き算、掛け算などの計算ができる
  • 文字: 結合（文字と文字をつなげる）はできるが、足し算のような計算はできない

データ型を確認してみよう（サンプルコード）

Pythonでは、type()関数を使って変数のデータ型を確認することができます。

num = 100
text = "こんにちは"
is_active = True
price = 19.80

print(type(num))       # 出力: <class 'int'>
print(type(text))      # 出力: <class 'str'>
print(type(is_active)) # 出力: <class 'bool'>
print(type(price))     # 出力: <class 'float'>

このように、type()関数を使えば、変数がどのデータ型なのかが一目で分かります。

まとめ：変数は「箱」、データ型は「箱の中身の種類」

  • データ型: 箱の中に入っているデータがどんな「種類」なのか
