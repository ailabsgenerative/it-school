# Pythonのfor/whileループと繰り返しの制御を学ぼう

## はじめに

プログラミングをしていると、「同じ処理を何度も繰り返したい」という場面に頻繁に出会います。例えば、リストの各要素を順番に処理したり、特定の条件が満たされるまで処理を続けたりする場合です。そんなときに活躍するのが「ループ（繰り返し）処理」です。

Pythonには、主に `for` ループと `while` ループという2つの強力なループ構文が用意されています。これらの使い方をマスターすれば、コードをより簡潔に、効率的に書くことができるようになります。

この記事では、`for` と `while` の基本的な使い方から、`break` や `continue` を使ったループの制御方法まで、サンプルコードを交えながら分かりやすく解説します。

---

## forループ：決まった回数や要素の数だけ繰り返す

`for` ループは、リストや文字列、あるいは `range()` 関数で生成される数値の列など、複数の要素を持つデータ（イテラブルオブジェクト）から要素を1つずつ取り出して処理を繰り返すのに便利です。

### 基本的な構文

```python
for 変数 in イテラブルオブジェクト:
    # 繰り返したい処理

range() を使った繰り返し

決まった回数だけ処理を繰り返したい場合に最もよく使われるのが range() 関数です。

# 0から4まで5回繰り返す
print("range(5)の場合:")
for i in range(5):
    print(f"現在の数値は {i} です")

# 1から5まで繰り返す
print("\nrange(1, 6)の場合:")
for i in range(1, 6):
    print(f"現在の数値は {i} です")

リストの要素を使った繰り返し

リストやタプルの各要素に対して同じ処理を行いたい場合に for ループは真価を発揮します。

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(f"I like {fruit}!")

whileループ：特定の条件が満たされるまで繰り返す

while ループは、指定した条件式が True の間、処理を繰り返し実行します。繰り返しの回数が事前に決まっていない場合に特に有効です。

基本的な構文

while 条件式:
    # 繰り返したい処理
    # (条件式を変化させる処理も忘れずに)

カウンターを使った繰り返し

カウンター変数を使い、その値が特定の数になるまで処理を繰り返す例です。

count = 0
while count < 5:
    print(f"現在のカウントは {count} です")
    count += 1  # カウントを1増やす (これを忘れると無限ループになる！)

print("ループが終了しました。")

ループの制御：break と continue

ループの途中で処理の流れをコントロールしたい場合があります。そのために break と continue という2つの命令が用意されています。

break：ループを中断する

break は、ループ処理をその場で完全に終了させたいときに使います。特定の条件が満たされたら、それ以上ループを続ける必要がない場合に使用します。

# 1から9までの数値で、5を見つけたらループを抜ける
for i in range(1, 10):
    if i == 5:
        print("5を見つけたので、ループを中断します。")
        break
    print(i)

continue：現在の回の処理をスキップする

continue は、現在の回の処理だけをスキップして、次の回の処理に進みたいときに使います。例えば、特定の条件に一致する要素だけ処理を省略したい場合などに便利です。

# 1から9までの数値で、偶数だけをスキップして奇数のみ表示する
for i in range(1, 10):
    if i % 2 == 0:  # 偶数かどうかを判定
        continue  # 偶数なら、この後のprint処理をスキップして次のループへ
    print(f"奇数です: {i}")

まとめ

今回は、Pythonにおける繰り返し処理の基本である for ループと while ループについて学びました。

  • for ループ: リストの要素数や range() で指定した回数など、繰り返しの数が決まっている場合に使うと便利です。
  • while ループ: ある特定の条件が満たされるまで処理を続けたい場合に適しています。
  • break: ループを完全に中断したいときに使います。
  • continue: 現在の回の処理だけをスキップしたいときに使います。

ループ処理は、プログラミングの基本であり、あらゆる場面で応用できる非常に重要な概念です。まずは簡単なコードからで良いので、実際に手を動かして、その動作を確かめてみてください。 ```
