Pythonの条件分岐(if/elif/else)をマスターしよう！

はじめに

プログラミングをしていると、「もしAという状況なら、Xという処理をする。そうでなければYという処理をする」といったように、特定の条件によって処理を分けたい場面がよくあります。これを実現するのが「条件分岐」です。

Pythonでは、if、elif、elseといったキーワードを使って、シンプルかつ直感的に条件分岐を記述することができます。この記事では、これらの基本的な使い方を、初心者の方にも分かりやすく解説していきます。

if文：もし〜ならば

最も基本的な条件分岐がif文です。「もし条件式が真（True）ならば、特定の処理を実行する」という働きをします。

書き方:

if 条件式:
    # 条件式が真(True)のときに実行される処理

ポイント: * ifの後ろには、真(True)か偽(False)を判定できる「条件式」を書きます。 * 条件式の最後にはコロン(:)を忘れないようにしましょう。 * 実行したい処理は、if文のブロック内にインデント（字下げ）して記述します。

サンプルコード:

age = 20

if age >= 18:
    print("あなたは成人です。")

このコードでは、変数ageが18以上なので、条件式age >= 18が真(True)となり、print関数が実行されて「あなたは成人です。」と表示されます。

else文：そうでなければ

if文の条件が満たされなかった（偽/Falseだった）場合に、別の処理を実行したいときにelse文を使います。

書き方:

if 条件式:
    # 条件式が真(True)のときに実行される処理
else:
    # 条件式が偽(False)のときに実行される処理

サンプルコード:

age = 15

if age >= 18:
    print("あなたは成人です。")
else:
    print("あなたは未成年です。")

今度は変数ageが15なので、条件式age >= 18は偽(False)となります。そのため、elseブロック内の処理が実行され、「あなたは未成年です。」と表示されます。

elif文：もし〜でなく、もし〜ならば

3つ以上の条件で処理を分けたい場合は、elif（else ifの略）を使います。elifを使うと、複数の条件を順番にチェックしていくことができます。

書き方:

if 条件式1:
    # 条件式1が真(True)のときに実行される処理
elif 条件式2:
    # 条件式1が偽(False)で、条件式2が真(True)のときに実行される処理
else:
    # すべての条件式が偽(False)のときに実行される処理

ポイント: * elifはifとelseの間に、いくつでも追加することができます。 * 上から順番に条件がチェックされ、最初に真(True)になったブロックの処理だけが実行されます。

サンプルコード:

score = 85

if score >= 90:
    print("評価はAです。素晴らしい！")
elif score >= 80:
    print("評価はBです。良い調子！")
elif score >= 70:
    print("評価はCです。もう少し頑張ろう！")
else:
    print("評価はDです。再挑戦が必要です。")

# 実行結果: 評価はBです。良い調子！

この例では、score >= 90は偽(False)ですが、次のscore >= 80が真(True)となるため、elifブロック内の処理が実行され、そこで条件分岐は終了します。

まとめ

今回は、Pythonにおける条件分岐の基本であるif、elif、elseの使い方を学びました。

  • if: 基本の条件分岐。条件がTrueなら処理を実行。
  • else: ifの条件がFalseだった場合に処理を実行。
  • elif: 複数の条件を順番にチェックしたいときに使用。
