# Pythonの宝箱！標準ライブラリの定番機能を使ってみよう

## 1. はじめに

Pythonをインストールすると、実はたくさんの便利な「お道具箱」が一緒に付いてきます。それが「標準ライブラリ」です。標準ライブラリは、Pythonに初めから組み込まれているモジュール（機能のまとまり）の集まりで、別途インストールすることなく、誰でもすぐに利用できます。

数学計算、ランダムな数値の生成、日付や時刻の操作、ファイルシステムの操作など、プログラミングでよく必要とされる機能が豊富に揃っています。この記事では、数ある標準ライブラリの中から、特に利用頻度が高く、初心者が覚えておくと便利な定番機能について、具体的な使い方をサンプルコードと共に解説します。

## 2. 具体的な使い方や概念の説明

標準ライブラリの機能を使うには、`import`というキーワードを使って、利用したいモジュールをプログラムに読み込むことから始めます。

```python
import モジュール名

それでは、具体的なモジュールを見ていきましょう。

mathモジュール: 数学計算の達人

平方根や三角関数など、少し高度な数学計算を行いたいときに便利なのがmathモジュールです。

サンプルコード:

import math

# 平方根を求める (√16)
result_sqrt = math.sqrt(16)
print(f"16の平方根は: {result_sqrt}")

# 円周率π
pi_value = math.pi
print(f"円周率は: {pi_value}")

# 半径5の円の面積を計算
radius = 5
area = radius * radius * math.pi
print(f"半径{radius}の円の面積は: {area}")

randomモジュール: 偶然をプログラムする

ゲームのサイコロのようにランダムな数値を生成したり、リストからランダムに要素を選んだりしたい場合にrandomモジュールが役立ちます。

サンプルコード:

import random

# 1から6までのランダムな整数を生成（サイコロ）
dice_roll = random.randint(1, 6)
print(f"サイコロの目は: {dice_roll}")

# リストからランダムに要素を1つ選ぶ（おみくじ）
fortunes = ["大吉", "中吉", "小吉", "凶"]
my_fortune = random.choice(fortunes)
print(f"今日の運勢は: {my_fortune}")

datetimeモジュール: 日付と時刻を自在に操る

現在の日付や時刻を取得したり、特定の日付を表現したり、日付の計算を行ったりするにはdatetimeモジュールが不可欠です。

サンプルコード:

import datetime

# 現在の日付と時刻を取得
now = datetime.datetime.now()
print(f"現在の日時: {now}")

# 日付と時刻を分かりやすい形式で表示
formatted_time = now.strftime("%Y年%m月%d日 %H:%M:%S")
print(f"フォーマット後の日時: {formatted_time}")

# 今日の日付だけを取得
today = datetime.date.today()
print(f"今日の日付: {today}")

osモジュール: コンピュータと対話する

ファイルやディレクトリ（フォルダ）を操作するなど、オペレーティングシステム（OS）の機能を利用したいときに使うのがosモジュールです。特に、ファイルパスを扱うos.pathは非常によく使われます。

サンプルコード:

import os

# カレントディレクトリ（今いる場所）のファイル一覧を取得
print("カレントディレクトリのファイル一覧:")
for file_name in os.listdir("."):
    print(file_name)

# OSに依存しない安全なファイルパスの作成
# "data"フォルダの中の"file.txt"を指すパスを作る
file_path = os.path.join("data", "file.txt")
print(f"作成されたパス: {file_path}")

jsonモジュール: データ交換の標準形式を扱う

json（ジェイソン）は、異なるプログラム間でデータをやり取りするためによく使われる、軽量なデータ形式です。Pythonの辞書やリストをJSON形式の文字列に変換したり、その逆を行ったりできます。

サンプルコード:

import json

# Pythonの辞書データ
user_data = {
    "id": 101,
    "name": "Taro Yamada",
    "email": "taro@example.com"
}

# 辞書をJSON形式の文字列に変換
json_string = json.dumps(user_data, indent=2, ensure_ascii=False)
print("--- JSON文字列 ---")
print(json_string)

# JSON形式の文字列をPythonの辞書に変換
parsed_data = json.loads(json_string)
print("--- Pythonの辞書 ---")
print(f"名前: {parsed_data['name']}")

3. まとめ

この記事では、Pythonの標準ライブラリの中から、以下の5つの基本的なモジュールを紹介しました。

  • math: 数学計算を簡単にする
  • random: ランダムな値を作り出す
  • datetime: 日付や時刻を扱う
  • os: ファイルやディレクトリを操作する
  • json: 他のプログラムとデータを交換する

標準ライブラリには、ここで紹介した以外にも数多くの便利なモジュールが存在します。公式ドキュメントを眺めて、「こんなこともできるんだ！」と発見するのもPython学習の楽しみの一つです。まずは今回学んだ機能を使いこなし、Pythonプログラミングの効率と楽しさを実感してみてください。 ```
